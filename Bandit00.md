# Bandit - Level 00 -> 01

## Objetivo
Descobrir a senha dentro do arquivo "readme"

## Comandos Recomendados / Utilizados
- Recomendados: ls (list), cd (change directory), cat (concatenate), file, du (disk usage), find
- Utilizados: ls, cat

## Resolução do Desafio
O primeiro passo que tomei depois de me conectar à máquina por SSH foi dar um ls para listar os arquivos e diretórios nela. Como se trata do nível 1, naturalmente a resposta foi dada de bandeja com o arquivo "readme" já na home do meu usuário. Usamos o cat para ler o arquivo e a senha está ali.
"The password you are looking for is: **6y2kwnwK6grgvwvpvLaa2T1cpFEKOhNR**"

## Conceitos Aprendidos
- ls lista arquivos no diretório atual
- cat lê e exibe o conteúdo de um arquivo
- Após conectar via SSH, o ambiente é igual ao terminal local

## Dificuldades
Nenhuma grande dificuldade, justamente por ser um nível introdutório.
