# OverTheWire - Bandit
Minha progressão nos níveis do [OverTheWire - Bandit](https://overthewire.org/wargames/bandit/), com anotações, comandos usados e raciocínio para chegar em cada solução. Todo o processo é feito por mim mesmo, só uso a IA na documentação dos níveis para gramática e formatação.

## O que é o Bandit?
O Bandit é um wargame: um jogo que te força a desenvolver habilidades para resolver cada nível. Cada nível tem uma senha escondida em que o objetivo é encontrar ela para ir para o próximo nível. 
O OverTheWire - Bandit te força a desenvolver comandos Linux dos mais básicos aos mais difíceis. O Bandit tem várias categorias: o Bandit é focada em Linux, mas também tem o Natas que foca em cibersegurança (que pretendo fazer no futuro).
